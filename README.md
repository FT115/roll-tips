# 滚动提词器

全屏滚动提词器，适用于演讲、录制、直播等场景。文字自下而上平滑滚动，支持中文自动换行。

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 把提词内容写入 words.txt

# 3. 运行
python teleprompter.py
```

## 操作

| 按键 | 功能 |
|------|------|
| `ESC` / `Q` | 退出程序 |
| `空格` | 暂停 / 继续滚动 |

## 流程

运行脚本 → 等待 `DELAY_BEFORE_START` 秒 → 文字从屏幕底部滚入 → 全部滚出顶部 → 停留 `WAIT_AFTER_END` 秒 → 自动退出。

## 配置

所有参数在 [config.py](config.py) 中修改，修改后无需重启即可生效（重新运行脚本）。

### 显示

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `FULLSCREEN` | `True` | 是否全屏 |
| `WINDOW_WIDTH` | `1280` | 窗口宽度（仅非全屏） |
| `WINDOW_HEIGHT` | `720` | 窗口高度（仅非全屏） |
| `BACKGROUND_COLOR` | `(0, 0, 0)` | 背景色 RGB |
| `TEXT_COLOR` | `(255, 255, 255)` | 文字色 RGB |

### 字体

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `FONT_SIZE` | `70` | 字体大小（像素） |
| `FONT_PATH` | `None` | 字体路径，`None` 自动查找微软雅黑/黑体 |
| `LINE_SPACING` | `20` | 行间距（像素） |
| `MARGIN_LEFT` | `120` | 左边距 |
| `MARGIN_RIGHT` | `120` | 右边距 |
| `MARGIN_TOP` | `80` | 顶部留白 |
| `MARGIN_BOTTOM` | `80` | 底部留白 |

### 滚动

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `SCROLL_SPEED` | `40` | 滚动速度（像素/秒），建议 30~120 |

### 时间

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `DELAY_BEFORE_START` | `2.0` | 开始滚动前等待（秒） |
| `WAIT_AFTER_END` | `3.0` | 结束后停留（秒） |

### 文本源

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `WORDS_FILE` | `"words.txt"` | 提词文本文件路径 |
| `FILE_ENCODING` | `"utf-8"` | 文本编码 |

## 依赖

- Python ≥ 3.6
- pygame ≥ 2.0
